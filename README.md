# DigiCanvas

To install dependencies:
```
npm install
```

To run in development mode:
```
npm run serve
```

# Deployment

For production (in Github Pages) run:
```
npm run deploy:prod
```

For test releases (in Netlify) run:
```
VERSION=X.Y npm run deploy:uat
```
NB: Substitute `X.Y` with the version to be deployed
